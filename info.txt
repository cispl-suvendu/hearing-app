Features for Admin:
1. Manage Categories and Subcategories: Add and organize quiz topics.
2. Add Questions: Create questions or upload them in bulk using a CSV file.
3. Create Exams: Combine questions into quizzes.
4. Assign Exams: Send quizzes to users via email.
5. Review Exams: Check and evaluate completed quizzes.
6. Manage Admin Users: Add other admins to help manage the platform.

nextjs, react, tailwind, ts, mongodb, html
